# 📱 Sistema de Controle de Atendimento

Este projeto foi desenvolvido como atividade da disciplina de Mobile, com o objetivo de simular um sistema de gerenciamento de filas de atendimento, semelhante ao utilizado em laboratórios e clínicas.

---

## 🎯 Objetivo

Criar um aplicativo utilizando Ionic + Angular capaz de:

* Emitir senhas de atendimento
* Organizar uma fila com prioridade
* Realizar chamadas de atendimento
* Registrar histórico com informações detalhadas

---

## ⚙️ Funcionalidades

* ✅ Geração de senhas:

  * SP (Prioritária)
  * SE (Exame)
  * SG (Geral)

* ✅ Fila de atendimento com regra de prioridade:

  * SP → SE/SG → SP → ...

* ✅ Chamada da próxima senha

* ✅ Histórico das últimas 5 senhas atendidas

* ✅ Exibição de:

  * Número da senha
  * Tipo
  * Guichê
  * Data e hora do atendimento

---

* Ionic

* TypeScript
